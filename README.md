# NexisPortfolio

## Description
NexisPortfolio is a clean and responsive personal portfolio built with HTML and CSS. It showcases projects, skills, services, and contact information. This project does not use JavaScript, making it lightweight and easy to maintain.

## Features
- Fully responsive design for desktop, tablet, and mobile
- Clean layout using Flexbox and CSS Grid
- Sections: Landing, Services, Portfolio, About, Stats, Skills, Contact, Footer
- CSS variables for easy customization of colors and spacing

## Installation
1. Clone or download the repository
2. Open `index.html` in any web browser

## Notes
- Interactive elements like sliders or hamburger menus are static due to no JavaScript
- Customize colors and fonts via the `:root` CSS variables in `master.css`
